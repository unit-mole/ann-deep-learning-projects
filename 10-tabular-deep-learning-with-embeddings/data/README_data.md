# Data Guide

This project uses a **deterministic synthetic dataset** generated by
`src/data_generation.py`. No private customer, employee, product, or company data is
included.

## Included file

- `sample_input.csv`: ten model-ready rows for testing the Streamlit batch prediction workflow.

## Recreate the full dataset

```bash
python -c "from src.data_generation import generate_synthetic_tabular_data; generate_synthetic_tabular_data().to_csv('data/full_dataset.csv', index=False)"
```

`data/full_dataset.csv` is intentionally excluded by `.gitignore` because it can be
reproduced from code and is not needed by the hosted application.

## Schema

Categorical features: `region`, `education_level`, `occupation`, `channel`,
`device_type`, `membership_tier`, and `product_category`.

Numerical features: `age`, `years_experience`, `monthly_sessions`,
`avg_basket_value`, `days_since_last_activity`, `credit_utilization`, and
`income_estimate`.

Target: `target`, a synthetic binary positive-outcome propensity label. The target
is for architecture demonstration and should not be interpreted as a real business
risk or eligibility decision.
