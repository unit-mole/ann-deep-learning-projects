Built and deployed a TensorFlow handwritten digit classifier using a tuned dense ANN, achieving 98.24% MNIST test accuracy and delivering an end-to-end inference pipeline with image normalization, confidence ranking, misclassification analysis, and Streamlit visualization.

## LinkedIn project description

Developed an end-to-end handwritten digit recognition application using TensorFlow/Keras and MNIST. The project includes a tuned dense neural network, custom image preprocessing for real-world uploads, per-class evaluation, confusion-matrix analysis, automated tests, and an interactive Streamlit interface that displays the predicted digit, confidence, runner-up class, and full probability distribution.
